# Monster Throne

Simulation-first 2D RPG о попаданце первого уровня в мире мега-монстров.

Текущий этап: **World Laboratory v0.3**.

Главный принцип проекта: сначала создать автономный мир, который способен жить без игрока. Существа охотятся, объединяются в стаи и расы, защищают территории, отступают, поедают туши, усиливаются, погибают и возвращаются слабее. Игрок будет добавлен после проверки основы экосистемы.

## Запустить лабораторию

Откройте `index.html` через локальный сервер:

```bash
npm run serve
```

Затем откройте:

```text
http://localhost:8080/
```

После принятия этой версии в `main` лаборатория будет развёрнута через GitHub Pages.

## Что реализовано в v0.3

- детерминированный seed;
- карты площадью `1×`, `2×` и `4×`;
- пространственная сетка вместо полного перебора всех существ;
- четыре экспериментальные расы и территории;
- генералы уровней 100, 250, 500 и 750;
- стайная охота слабых существ;
- индивидуальное поведение и оценка риска у существ 500+;
- предпочтение добычи чужой расы;
- несмертельные внутривидовые дуэли;
- распад стаи после тяжёлых потерь;
- перерождение возле территории своей расы;
- свежие, испорченные и разложившиеся туши;
- постоянное усиление и неограниченный уровнем временный эффект от свежего мяса вершины;
- ускорение до `×100` с более редкой отрисовкой;
- быстрые браузерные проверки в Web Worker;
- автоматические системные и сценарные тесты.

## Метки статуса

- `[DECISION]` — принятое правило;
- `[EXPERIMENT]` — временная реализация или коэффициент;
- `[AI PROPOSAL]` — предложение ассистента, не часть авторского видения без подтверждения;
- `[OBSERVATION]` — результат теста.

Все нынешние названия рас, численность, размеры территорий и коэффициенты поведения остаются экспериментальными.
