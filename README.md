# Monster Throne — simulation-first 2D RPG prototype

Черновой репозиторий системной 2D RPG о попаданце в мире мега-монстров.

Главная идея: сначала создаётся автономный мир, в котором существа охотятся, умирают, перерождаются слабее, пожирают трупы, мутируют и стремятся к центру карты. Игрок будет добавлен только после того, как мир научится жить без него.

## Что уже есть

- хранилище исходных идей без попытки преждевременно всё «причесать»;
- журнал принятых решений;
- открытые вопросы, которые ещё нельзя считать правилами;
- первая браузерная симуляция без игрока;
- свежесть трупов: свежий, испорченный, останки;
- большие туши, которые могут одновременно поедать несколько существ;
- охота, бегство при низком здоровье, поедание, рост доминирующей характеристики;
- потеря 10% характеристик после смерти и последующее перерождение;
- неподвижный Король демонов в центре карты;
- ускорение симуляции до ×100 и выбор существ для просмотра параметров.

## Как открыть первую симуляцию

Откройте файл:

```text
simulator/index.html
```

Он не требует установки программ или интернета.

Кнопки сверху:

- **Пауза** — остановить мир;
- **×1 / ×5 / ×20 / ×100** — скорость времени;
- **Сбросить мир** — заново создать мир с тем же набором правил;
- клик по существу — показать его характеристики и историю.

## Структура

```text
docs/00-idea-vault.md           сырые идеи, которые нельзя потерять
docs/DECISIONS.md               уже принятые решения
docs/OPEN-QUESTIONS.md          вопросы и противоречия
docs/ROADMAP.md                 порядок разработки
docs/systems/                   рабочие описания отдельных систем
data/simulation-config.json     первые числовые настройки
simulator/                      первая автономная симуляция
```

## Важный принцип

`00-idea-vault.md` не переписывается задним числом. Новые мысли добавляются туда отдельными датированными блоками. Рабочие правила формируются в `docs/systems`, а подтверждённые решения фиксируются в `DECISIONS.md`.

Это позволяет одновременно двигаться к рабочей игре и не терять первоначальную нить идеи.
