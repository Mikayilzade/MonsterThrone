# Monster Throne — simulation-first 2D RPG

Системная 2D RPG о попаданце в мире мега-монстров.

Герой начинает на первом уровне, тогда как даже обычные насекомые намного сильнее него. Он выживает с помощью наблюдения, ловушек, готовки, алхимии и тактической паузы, поглощает характеристики и свойства монстров и постепенно движется к Королю демонов 1000-го уровня.

Главный принцип разработки: сначала создать автономный мир, который способен жить, охотиться, воевать, мутировать и менять лидеров без игрока. Игрок добавляется после проверки основы экосистемы.

## Правило сохранения идей

`docs/00-idea-vault.md` не переписывается задним числом ради красивой версии проекта. Новые мысли добавляются отдельными датированными блоками.

После обсуждения:

1. рабочая версия правила переносится в `docs/systems/`;
2. подтверждённое направление фиксируется в `docs/DECISIONS.md`;
3. нерешённые противоречия остаются в `docs/OPEN-QUESTIONS.md`.

Так можно одновременно развивать код и не терять первоначальную нить игры.
