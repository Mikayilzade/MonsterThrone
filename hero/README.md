# Monster Throne — «Путь слабого» v0.5 demo 1

Первая игровая демоверсия от лица героя. Это уже не экран наблюдения за всем миром: камера следует за персонажем, а автономная экосистема продолжает жить вокруг него.

## Главное правило героя

Герой навсегда связан с неразрушимым артефактом **«Якорь вечной души»**.

- целостность души всегда `100%`;
- окончательная смерть невозможна;
- после гибели герой получает новое тело возле лагеря;
- постоянные характеристики, навыки, вещи и история сохраняются;
- рассеивается только временная сила от свежей плоти.

## Что входит в демо

- большой идеально круглый остров площадью примерно в десять раз больше первой лаборатории;
- локальная камера 2D сверху, разные биомы и настоящая морская граница;
- управление на клавиатуре/мыши и сенсорные элементы для телефона;
- копьё, рывок с неуязвимостью, скрытность в лесу и ловушки;
- самостоятельные травоядные, хищники, падальщики и редкий молодой верховный хищник;
- убийства между монстрами и возможность украсть результат чужой охоты;
- свежие туши, скрытый постоянный потенциал и сильный временный бонус;
- уровень, созревающий за минуту при открытом потолке;
- ресурсы, голод, лечение, лагерь и готовка;
- цепочка начальных целей, свободное исследование и локальное сохранение;
- тактическая пауза с упрощённой постановкой точки движения.

## Запуск

Из корня репозитория:

```bash
py -m http.server 8080
```

Открыть:

```text
http://localhost:8080/hero/
```

После обновления файлов нажать `Ctrl + F5`.

## Управление на ПК

- `WASD` / стрелки — движение;
- `ЛКМ` / `Space` — удар копьём;
- `Shift` — рывок;
- `E` — взаимодействие;
- `Q` — ловушка;
- `R` — использовать еду;
- `P` / `Escape` — тактическая пауза;
- `J` — журнал;
- колесо мыши — масштаб;
- правая кнопка — автоматическое движение к точке.

## Локальная проверка

- `17/17` системных проверок;
- `15/15` проверок в трёх двухчасовых headless-сценариях;
- целостность души героя всегда оставалась `100%`;
- все существа оставались внутри круглого острова;
- не обнаружено бесконечного накопления туш или выхода за морскую границу.

Тестовый стенд запускался локально. GitHub Actions не добавлялись и не запускались.

## Ограничения demo 1

- это вертикальный срез внешней части острова, а не полная игра до Трона;
- графика процедурная, без финальных нарисованных ассетов;
- полноценной очереди действий на паузе пока нет — только точка движения;
- нет диалогов, питомцев, поселений, сложного крафта и магии;
- дальняя часть мира симулируется проще, чем область рядом с героем;
- баланс рассчитан для первых часов и обязательно будет меняться после ручной игры.
