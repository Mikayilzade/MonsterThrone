# World Laboratory v0.4 alpha 1

Первый живой вертикальный срез принятой модели мира.

## Запуск

Из корня репозитория:

```bash
npm run serve
```

Открыть:

```text
http://localhost:8080/v04/
```

## Локальные тесты

```bash
npm run test:v04
npm run test:v04:scenarios
npm run test:v04:all
```

Тесты не подключены к автоматическому запуску GitHub Actions.

## Что наблюдать

- держатся ли виды и не возникает ли одна доминирующая стратегия;
- почему существа находятся у потолка;
- достаточно ли заметны туши и временная сила;
- не слишком ли быстро повреждаются души;
- естественны ли охота, бегство и питание;
- появляются ли потомки, мутации и именные существа;
- понятно ли различие уровня, потолка, скрытого потенциала и временных характеристик.
