# World Laboratory v0.4 alpha 2

Первый живой вертикальный срез принятой модели мира, переработанный после ручной проверки автора.

Главная цель alpha 2 — не добавить новые механики, а сделать автономный мир понятным наблюдателю.

## Запуск

Из корня репозитория:

```bash
py -m http.server 8080
```

или:

```bash
npm run serve
```

Открыть:

```text
http://localhost:8080/v04/
```

После обновления файлов использовать `Ctrl + F5`, чтобы браузер не оставил старую alpha 1 в кэше.

## Что изменилось в alpha 2

- клик выбирает ближайшее существо в увеличенной области, а не требует попадания в маленькую точку;
- добавлены наведение, краткая карточка и заметная подсветка выбранного существа;
- карта поддерживает приближение, отдаление, перетаскивание и слежение;
- растения, туши и существа получили разные формы и понятную легенду;
- подписи можно показывать для важных, выбранного или всех существ;
- состояния, причины решений и причины смерти переведены на русский;
- инспектор разделён на вкладки `Главное / Развитие / Тело / Навыки / Жизнь`;
- сверху показывается человеческое объяснение: что существо делает и почему;
- маршрут к текущей цели отображается пунктиром, когда ядро предоставляет координаты цели;
- события мира получили фильтры;
- добавлено объяснение назначения стенда при первом запуске;
- существа за последней окружностью подсвечиваются красным и считаются диагностической проблемой;
- интерфейс лучше перестраивается на узких экранах.

## Локальные тесты

```bash
npm run test:v04
npm run test:v04:scenarios
npm run test:v04:ui
npm run test:v04:all
```

UI-тесты проверяют перевод, поиск ближайшей точки, объяснение поведения и диагностику внешней границы. GitHub Actions не подключены.

## Что наблюдать

1. Выбрать существо и начать с скорости `×1` или `×5`.
2. Проверить, понятно ли без дополнительных объяснений, что оно делает и почему.
3. Посмотреть вкладку развития: текущий уровень, потолок, скрытый потенциал и блокировки.
4. Включить слежение и проследить отдельную судьбу до серьёзного события.
5. Проверить красную диагностику внешней границы.
6. После этого использовать `×20 / ×100` для длинных изменений популяции.

## Ограничения

- alpha 2 улучшает наблюдение, но не превращает стенд в готовую игру;
- поведение групп, территории, бой и разнообразие мутаций остаются ранними;
- внешний край пока диагностируется и подсвечивается, но причина выхода существ требует отдельной проверки ядра;
- переводы покрывают известные состояния, неизвестный новый внутренний термин будет показан в упрощённом виде;
- сохранение мира в файл ещё не реализовано.
