Дата завершения: 19.07.2026

- [x] детерминированное ядро мира;
- [x] площади карты `1× / 2× / 4×`;
- [x] пространственная сетка;
- [x] расы, территории и генералы 100/250/500/750;

- [x] межрасовый приоритет и несмертельные внутренние дуэли;

- [x] перерождение с потерей 10% характеристик;

- [x] headless-прогоны и Web Worker для быстрых проверок.

## Проверка

- [x] 19/19 системных тестов;
- [x] 8/8 сценарных критериев;
- [x] 15 часовых прогонов: по 5 seed на каждом размере карты;
- [x] payload-файлы ветки сверены с локально проверенной версией по Git blob SHA;
- [x] синтаксис распакованных `core` и `app` проверен;
- [x] CI переведён в ручной режим, чтобы промежуточные коммиты не создавали поток писем.

## После принятия PR

- GitHub Pages разворачивает лабораторию из `main`;
- ручная проверка проводится по `docs/v0.3/MANUAL-TEST.md`;
- новые идеи сначала добавляются в `docs/00-idea-vault.md` или с меткой `[AI PROPOSAL]`.

## Ограничения этапа

- баланс смертности не считается финальным;
- почти все туши съедаются до порчи;
- названия рас, популяции, территории и коэффициенты являются `[EXPERIMENT]`;
- игрок, ловушки, магия, готовка и полноценный крафт ещё не добавлены.
