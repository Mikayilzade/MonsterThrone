# Шаблон отчёта о тесте

Скопируй блок ниже в новое сообщение или GitHub Issue и заполни только то, что успел заметить.

```text
Версия симуляции:
Дата:
Seed, если показан:
Скорость:
Реальное время теста:
Время мира в момент наблюдения:

Живых существ:
Смертей:
Перерождений:
Свежих туш:
Испорченных туш:
Останков:
Сильнейший уровень:

Что происходило:

Где на карте:

Какие существа участвовали:

Что ожидалось:

Что показалось странным:

Повторилось ли после сброса мира:

Нагрузка ноутбука:
- CPU Chrome:
- общая CPU:
- память Chrome:
- шум вентиляторов: низкий / средний / сильный

Скриншот или видео:
```

Не обязательно заполнять всё. Наиболее полезны версия, скорость, время мира, место на карте и конкретное описание поведения.
